### Mise en ligne
1. S’assurer que le dépôt `alphapdf-privacy` est **public**
2. Aller dans **Settings → Pages**
3. Choisir **Deploy from branch**
4. Sélectionner la branche **`main`**
5. Sélectionner le dossier **`/ (root)`**
6. Cliquer sur **Save**
7. Attendre quelques minutes pour la mise en ligne ou la mise à jour du site
