Politique de confidentialité (RGPD)

Éditeur : Amine Telecom / Noeva Labs. Cette politique s’applique à toutes nos applications publiées sur Google Play. Les spécificités par application (permissions, publicité, achats) sont indiquées dans la page Nos applications.

1) Données traitées

  • Données locales uniquement : préférences d’affichage (zoom, thème), derniers fichiers ouverts. Ces informations restent sur l’appareil.
  • Aucune transmission de vos fichiers PDF à nos serveurs.
  • Analytique : uniquement si tu ajoutes un SDK (ex : Google Analytics via Tag). Par défaut, aucun tracking n’est activé.

2) Publicité et consentement

Lorsque la publicité est activée dans une app, nous utilisons Google AdMob. En Europe, le module UMP (consentement) peut s’afficher. Une option Sans pub (achat unique) peut être proposée selon l’app — voir la fiche de l’app concernée.

3) Base légale

  • Exécution du service : ouvrir et afficher vos PDF.
  • Intérêt légitime / Consentement : affichage de publicité (selon les régions et votre choix via UMP).

4) Vos droits

Vous pouvez désinstaller l’app à tout moment : toutes les données locales sont supprimées. Pour toute demande RGPD (accès, effacement, opposition), contactez : support@alphapdf.com.

5) Sécurité

Les données utilisateur sont stockées localement sur votre appareil. Aucun transfert vers un serveur privé. Nous limitons les permissions au strict nécessaire indiqué dans chaque fiche d’app.
