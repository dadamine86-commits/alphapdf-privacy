🌐 Déploiement sur GitHub Pages


Étape 1 — Vérifier le dépôt




Ouvrir le dépôt GitHub alphapdf-privacy


Vérifier que le dépôt est public


Vérifier que tous les fichiers du site sont bien à la racine




Étape 2 — Vérifier GitHub Pages




Aller dans Settings


Ouvrir Pages


Vérifier les paramètres suivants :



Source : Deploy from a branch


Branch : main


Folder : / (root)






Enregistrer si nécessaire




Étape 3 — Publier une mise à jour




Modifier le fichier voulu


Enregistrer


Committer les changements sur main


Attendre quelques minutes


Ouvrir le site public pour vérifier

🔒 Conformité


Le site légal doit rester cohérent avec l’application réellement distribuée.


En particulier :




la politique de confidentialité doit rester cohérente avec Google AdMob


la politique de confidentialité doit rester cohérente avec Google UMP


les informations sur les achats intégrés doivent rester cohérentes avec Google Play


les pages légales ne doivent pas annoncer des éléments inexacts
